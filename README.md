# goit-js-hw-05

Iterative methods of arrays

## Tasks

### Task 1: Get User Names

- **File**: `js/task-1.js`
- **Description**: Implement a function `getUserNames` that takes an array of
  user objects and returns an array of user names using the `map` method.

### Task 2: Get Users with Friend

- **File**: `js/task-2.js`
- **Description**: Implement a function `getUsersWithFriend` that takes an array
  of users and a friend's name, and returns an array of users who have that
  friend in their friends list using the `filter` method.

### Task 3: Sort by Descending Friend Count

- **File**: `js/task-3.js`
- **Description**: Implement a function `sortByDescendingFriendCount` that takes
  an array of users and returns them sorted by the number of friends in
  descending order using the `toSorted` method.

### Task 4: Get Total Balance by Gender

- **File**: `js/task-4.js`
- **Description**: Implement a function `getTotalBalanceByGender` that takes an
  array of users and a gender, and returns the total balance of users of that
  gender using `filter` and `reduce` methods.

## How to Run

Open `index.html` in a browser to execute all tasks and see console outputs.
